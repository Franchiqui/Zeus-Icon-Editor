# PocketBase

Este directorio contiene instaladores para levantar un backend ligero usando PocketBase sin requerir instalaciones previas.

## Opciones por sistema operativo
### Windows
1. Abre PowerShell como Administrador
2. Navega a la carpeta pocket-base
3. Ejecuta: powershell -ExecutionPolicy Bypass -File install-pocketbase.ps1


El script descargará automáticamente la versión correcta de PocketBase para tu sistema y ejecutará el servidor.
